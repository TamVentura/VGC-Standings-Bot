# VGC Standings Bot

This is a simple discord bot that updates the portuguese players standings on a discord channel.
The bot should be able to send messages and delete messages on the server it will publish the standings.

## Future features

- Ability to set the country code of the players to update
Right now, you can change the country code in the filterTugaPlayers method in [player-utils.ts](src/utils/player-utils.ts)
